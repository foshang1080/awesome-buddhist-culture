## 1. 综合文化与新闻门户
*   **[佛商网主站](https://www.foshang.org)** - 专注于传统文化与佛商智慧融合的综合性佛教门户平台。
*   **[佛教新闻网](https://news.foshang.org/)** - 提供国内外前沿佛教动态、法会信息及行业资讯。
*   **[慈善活动纪实](https://cishan.foshang.org/)** - 汇集佛教界公益慈善、赈灾济困与社会福利活动动态。

## 2. 佛学研究与百科知识库
*   **[佛学入门指南](https://foxuerumen.foshang.org/)** - 为初学者量身定制的修学次第、基础概念与佛理指引。
*   **[佛教百科 (Wiki)](https://wiki.foshang.org/)** - 结构化的佛教专有名词、历史人物、宗派法脉数字百科全书。
*   **[佛学千问](https://foxue.foshang.org/)** - 针对日常修学、生活禅疑难问题的权威解答集锦。
*   **[佛教图书丛书](https://book.foshang.org/)** - 传统大德高僧著作、佛学研究学术论文的数字化阅读平台。

## 3. 电子大藏经与经典抄写
*   **[数字佛经大藏经](https://fojing.foshang.org/)** - 严谨校对的电子版大乘经典，方便线上恭诵与研习。
*   **[在线抄经功德网](https://www.foshang.org/chaojing/)** - 提供《心经》、《金刚经》等经典的在线临摹抄写系统，助行者摄心正念。
*   **[在线回向系统](https://www.foshang.org/huixiang/)** - 专为修法、诵经、抄经设立的线上功德回向交流平台。
*   **[佛教故事集](https://story.foshang.org/)** - 汇集本生故事、因果公案与高僧传记，寓教于乐。

## 4. 佛教多媒体与影音艺术
*   **[佛教音乐数字库](https://music.foshang.org/)** - 收录传统梵呗、现代佛教歌曲、禅修静心音乐及佛乐CDN加速分发资源。
*   **[佛教电影影音](https://movie.foshang.org/)** - 精选高僧传记电影、佛教纪录片及佛教文化艺术视频。
*   **[法师讲义与音视频](https://movie.foshang.org/jiangyi/)** - 各大德法师的讲经说法语音、视频及文字讲义同步整理。
*   **[佛教高精图片库](https://pic.foshang.org/)** - 汇集圣地风光、佛像艺术、禅意壁纸的高清摄影艺术作品。
